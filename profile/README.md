# Metaverse

重新构思、不同于类unix内核的现代微内核操作系统。

原生rust支持。

兼容libc以及posix接口。

## 获取

从仓库[kernel-release]获取最新版本的内核源码或可运行的内核镜像。

编译组织中的其它项目使其支持unix工具链、GNU工具链以及llvm工具链等。
